Architecture Overview

Version 2.0  ·  March 2026  ·  Confidential

1. Product Vision

FindCare is a conversational AI that helps users navigate healthcare — finding providers, understanding symptoms, researching clinical trials, and producing transcripts they can share with their care team. All of this happens within a single AI session using a ChatGPT / Claude.ai style interface. There is no separate "About Us" application; transparency about the product is one intent path within the same session.

Why CrewAI? Multi-LLM support — each agent can use the best model for its task (Claude for reasoning, GPT-4o-mini for cost-sensitive tasks, specialist models as available). The Anthropic Agent SDK is Claude-only and was not chosen for this reason.

The user never sees the routing. Every intent path stays within the same conversation thread. The session transcript is a first-class output — users can download and share it with their doctor.

6. Data and Privacy

HIPAA Consent Flow

Q1: "May we save a verbatim transcript of this conversation with your contact details?"
Yes → Store full chat history (PII intact, consent exchange included as evidence)
Q2 (if Q1 declined): "May we save a de-identified summary instead?"
Yes → LLM summarizes → deIdentify() scrubs PII → stored in notes field only
Neither consented: Contact fields only — no history stored
